# 신고해죠

## 누구의 어떤 문제를 푸는가

정보연계센터에 로그인해서 필드마다 수기 입력하고, 보험별로 다른 양식에 헷갈리고, 14일 기한에 쫓기는 반복 수작업을 없애준다.

## 핵심 동작 1개
관리하던 엑셀을 그대로 업로드하면,
4대보험 신고서 PDF를 자동 생성하고 관할 공단에 FAX까지 발송한다.

## 누구에게 보여줄 것인가
